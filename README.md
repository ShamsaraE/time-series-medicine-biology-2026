## Course Description

This practical course introduces modern time series analysis techniques
with applications in medicine and biology.

Topics include:

- Trend and seasonality
- Additive vs multiplicative dynamics
- Autocorrelation and partial autocorrelation
- Decomposition methods
- Forecasting models
- Biomedical case studies
